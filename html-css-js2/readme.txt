1、网页的组成部分：内容、表现、行为
    内容：网页里面主要展示的内容，一般用html实现；
    表现：网页的内容以什么样式展示，一般用css实现；
    行为：页面中元素与输入设备交互的响应，一般用javascript实现。

2、HTML:Hyper Text Markup Language超文本标记语言
    是通过标签来标记要显示的网页中积部分。网页文件本身是一种文本文件，通过在文本文件中添加标记符，
可以告诉浏览器如何显示其中的内容(如文字如何处理等)。
    不需要编译，由浏览器直接解析。
